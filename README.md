# EIC Management System

Projet EIC Management System - Gestion des membres et événements du club.

## Contenu du dépôt
- `docs/` : MCD, MLDR et documentation
- `sql/schema.sql` : script SQL pour créer la base de données

## Objectif
Ce projet permet de gérer les membres, les chefs de cellules, les événements et les contacts du club EIC.
